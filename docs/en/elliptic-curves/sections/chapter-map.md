## Chapter Map

Elliptic curves may look intimidating, but the mathematics you need is accessible: equations, slopes, and basic geometry.

By the end of this chapter you will be able to:

- Explain the general equation $y^2 = x^3 + ax + b$.
- Verify whether a point belongs to the curve.
- Add two distinct points and double a point.
- Handle the point at infinity as the neutral element of the group.
- Implement a `Point` class in Python.

In the next chapter we will combine these curves with finite fields to build ECDSA.
