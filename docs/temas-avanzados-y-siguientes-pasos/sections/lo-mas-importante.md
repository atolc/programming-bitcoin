## Lo mas importante

- El libro construye las piezas fundamentales: campos finitos, curvas, firmas, serializacion, transacciones, Script, bloques, red, SPV, filtros Bloom y Segwit.
- El siguiente salto es integrar esas piezas en software usable.
- Una wallet requiere manejo seguro de llaves, seleccion de UTXOs, calculo de comisiones, construccion de transacciones y experiencia de usuario clara.
- Lightning Network extiende Bitcoin con canales de pago y liquidacion eventual en cadena.
- Un explorador de bloques ayuda a entender datos reales: bloques, transacciones, UTXOs, scripts y direcciones.
- Una tienda web con pagos Bitcoin conecta el protocolo con casos de uso comerciales.
- Crear librerias propias puede consolidar conocimiento, siempre que se sea cuidadoso con seguridad criptografica.
- Contribuir a proyectos existentes exige leer codigo, entender pruebas y respetar reglas de consenso.
