## Chapter Map

Serialization is the bridge between abstract cryptography and the data that circulates on the Bitcoin network.

By the end of this chapter you will be able to:

- Encode public keys in SEC format (uncompressed and compressed).
- Serialize ECDSA signatures in DER format.
- Convert between bytes and Base58Check addresses.
- Understand endianness and why Bitcoin uses little-endian in many fields.
- Implement these conversions in Python.

In the next chapter we will use these tools to build and read complete transactions.
