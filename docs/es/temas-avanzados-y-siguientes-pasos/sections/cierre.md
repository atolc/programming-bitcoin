## Cierre

Has completado el recorrido de *Programming Bitcoin*: desde campos finitos y curvas elípticas hasta bloques, red P2P, SPV, filtros Bloom y SegWit.

Lo esencial de todo el viaje:

- Bitcoin combina **criptografía**, **estructuras de datos** y **incentivos económicos**.
- Cada capa (transacción → script → bloque → red) se apoya en la anterior.
- Implementar en Python obliga a entender bytes, no solo APIs de alto nivel.

El protocolo sigue evolucionando (Taproot, Lightning, covenant research), pero los fundamentos que construiste aquí no caducan. Sigue leyendo BIPs, ejecutando código en testnet y, cuando estés listo, contribuye de vuelta a la comunidad que mantiene este sistema descentralizado.
