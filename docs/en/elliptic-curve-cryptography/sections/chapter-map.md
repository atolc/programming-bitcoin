## Chapter Map

This chapter brings together finite fields and elliptic curves to build the cryptography Bitcoin uses every day.

By the end you will be able to:

- Work with elliptic curves over $F_p$ instead of the reals.
- Implement modular point addition and scalar multiplication.
- Describe the secp256k1 curve and its generator point $G$.
- Generate public keys from a private key.
- Sign and verify messages with ECDSA.
